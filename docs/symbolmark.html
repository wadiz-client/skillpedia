1. 개요

Skillpedia(스킬피디아)는 GitHub 저장소에 흩어진 SKILL.md 파일을 수집해 구조화된 문서로 제공하는 서비스입니다. 심벌마크는 이 핵심 가치, 곧 흩어진 스킬을 하나의 문서로 모으는 일을 가장 단순한 형태로 압축합니다.

두 개의 점(스킬)과 두 개의 선(문서)만으로 구성해, 브라우저 탭의 16px 파비콘부터 스플래시 화면까지 흔들림 없이 읽힐 수 있도록 설계했습니다.

3. 구성 원리 — w 리듬

심볼의 모든 치수를 하나의 원자 단위 w에서 이끌어냈습니다. w는 문서 라인의 굵기이며, 여기서 선·점·간격을 모두 w의 배수로 정했습니다. 잉크(획)와 공백을 1:1로 반복하는 이 ‘w 리듬’ 덕분에 어떤 크기에서도 밀도와 비례를 유지합니다.

점 지름 = 2w — 스킬 점의 지름은 선 굵기의 2배, 문서 줄보다 큰 무게로, 흩어져 있던 스킬이 문서를 이끄는 단위임을 나타냅니다.

모든 간격 = w — 선과 선 사이, 점과 선 사이 간격이 모두 선 굵기와 같은 w입니다. 전체는 5w × 5w의 정방형으로 맞물려 배경 없이 놓여도 안정적이고 어느 크기에서도 균질한 밀도를 유지합니다.

5. 색상

심벌마크는 흑백을 원칙으로 합니다. 밝은 배경에는 잉크 블랙(#191F28 또는 순수 #000), 어두운 배경에는 화이트(#FFF)를 사용합니다. 별도의 색을 입히거나 그라디언트를 적용하지 않습니다.

6. 보호 여백과 최소 크기

심볼 주위에는 최소 w만큼의 보호 여백(clearspace)을 확보해 다른 요소가 침범하지 않도록 합니다. 최소 사용 크기는 16px이며, 그 이하로는 형태가 뭉개지므로 사용하지 않습니다.
